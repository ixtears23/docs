
> 소스코드에 대한 문서를 따로 만들기보다 소스코드와 문서를 하나의 파일로 관리하는 것이 좋다.  
> 소스코드의 주석 `/**~*/`에 소스코드에 대한 정보를 저장하고,  
> 소스코드의 주석으로부터 HTML 문서를 생성해내는 프로그램(javadoc.exe)을 만들어서 사용했다. 

## 표준 애너테이션

> 자바에서 기본적으로 제공하는 애너테이션  

 - @Override
 - @Deprecated
 - @SuppressWarnings
 - @SafeVarargs
 - @FunctionalInterface
 - @Native

#### 표준 애너테이션 중 메타애너테이션

> 애너테이션을 정의하는데 사용되는 애너테이션

 - @Target
 - @Documented
 - @Inherited
 - @Retention
 - @Repeatable
