- webtob 홈 경로 확인
`> printenv`  

- webtob 버전확인  
`> wscfl -v`  

- webtob license 확인  
`> jeusadmin -licensedue`  

- webtob license 경로  
`WEBTOBDIR/license`  

- http.m 환경설정 파일 경로
`WEBTOBDIR/config`  

- http.m 컴파일 명령어(컴파일 하면 해당경로에 wsconfig 파일 생성 됨)  
`> wscfl -i http.m`  

- webtob 서버 중지/가동 명령어
`> wsdown`  
`> wsboot`
